We are going to use two different sets to for the columns and the rows which contain zeroes in the matrix. Then iterate the matrix and check if the current element is zero or not, if it is then store its row and column value in the set. Then get all the rows and columns from the set and update all the values of them to zero in matrix.

First, we traverse the matrix using two nested loops. For each cell (i, j) if it is 0 then mark all the elements of row i and col j as -1 except that contains zero. Then traverse the matrix and mark cell 0 which contains -1. We get our final matrix.
